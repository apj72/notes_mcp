# GitHub Repository Description

## Short Description (for GitHub repo page)

**Local MCP server for creating notes in Apple Notes on macOS with queue-based pull worker support**

## Longer Description (for README or About section)

A secure, local MCP (Model Context Protocol) server for creating notes in Apple Notes on macOS. Features include HMAC-authenticated queue processing via GitHub Gists, comprehensive security controls (rate limiting, folder allowlisting, confirmation mode), and audit logging. Supports both direct MCP tool calls and pull-based job processing.
